# Coding-lab_Group1
First Group

Hospital Management System Scripts
Project Overview
This project consists of a collection of Bash scripts designed to assist with hospital administration, data analysis, and archive management. The scripts automate common hospital operations such as system initialization, data security, patient vital processing, water usage auditing, and log archiving. The goal is to improve efficiency, maintain data integrity, and support hospital management through automation.

Project Structure
Script 1: hospital_admin.sh
Handles hospital administration tasks including:
System initialization
Data security implementation
Main execution workflow
Script 2: hospital_analysis.sh
Performs hospital data analysis tasks including:
Processing patient vital signs
Water usage auditing
Script 3: hospital_archive.sh
Manages archival operations including:
Log rotation and archive maintenance

Instructions
Shell Scripting: Build interactive scripts using select, case, and functions.
Permission Mastery: Apply chmod and chown to protect sensitive medical logs.
Data Analysis: Use grep and awk to filter emergencies and calculate facility averages.
Git Collaboration: Manage a shared codebase across 6 members using branches and conflict resolution.
